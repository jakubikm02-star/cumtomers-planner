# CUMTOMERS Planner

Pornhub-style weekly planner for technician trips (SWAN / OTRS).

## Features
- Týždenný plán výjazdov s časovými slotmi
- Kategórie farieb (5G / Optika / Jednoduchá / FTTB)
- Parser z Customers dumpu
- Plánovanie / Na zavolanie
- Automatické ukladanie do localStorage

## Live demo
https://jakubikm02-star.github.io/cumtomers-planner

## How to use
1. Otvor `index.html`
2. Pridávaj výjazdy manuálne alebo cez parser
3. Data sa ukladajú automaticky
